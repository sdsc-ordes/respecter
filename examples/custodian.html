The swiss data custodian ontology provides SHACL shapes and concepts (classes and properties) to model human and machine readable contracts.

Introduction

The swiss data custodian ontology provides SHACL shapes and concepts (classes and properties) to model human and machine readable contracts.

"This ontology and the tool ReSpecter is still in active development. See ReSpecter for more details on the tool."

Data Handling

Data Handling

Definition

A high-level Class to describe 'data handling'. This can consist of personal data being processed for a purpose, involving entities, using technical and organisational measures, applicable risks, rights, and legal basis.
