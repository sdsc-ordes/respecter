The swiss data custodian ontology provides SHACL shapes and concepts (classes and properties) to model human and machine readable contracts.

Introduction

The swiss data custodian ontology provides SHACL shapes and concepts (classes and properties) to model human and machine readable contracts.

"This ontology and the tool ReSpecter is still in active development. See ReSpecter for more details on the tool."
